Dans ce projet, la plupart du travail est effectué sur le node-red il suffit de taper les commandes :

sudo apt-get install nodejs           // installation 
node-red 
sudo npm install node-red-dashboard 
npm install node-red-openweathermap
sed -x 

Des commandes utiles:

node-red-start
node-red-stop
node-red-restart 
sed -x                        // pour le débogage
